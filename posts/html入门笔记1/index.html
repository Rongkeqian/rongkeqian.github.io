HTML 入门笔记

HTML 是由 Web 的发明者 Tim Berners-Lee 于 1990（我们一样的，我不能永恒，你应该可以）创立的一种标记语言。

HTML 的起手怎么写：图

常用的表章节的标签有哪些，分别是：

  • h1~h6 标题
  • section 章节
  • article 文章
  • P 段落
  • header 头部
  • footer 脚部
  • main 主要内容
  • aside 旁支内容
  • div 划分

全局属性有哪些

  • class 类
  • contenteditable 使元素可以编辑
  • hidden 隐藏
  • id 元素的唯一 id
  • style 元素的行内 CSS 样式
  • tabindex 元素的 tab 键次序
  • title 有关元素的额外信息

常用的内容标签

  • ol+li(ordered list+list item) 有序列表
  • ul+li(unordered list + list item) 无序列表
  • dl+dt+dd(description list + item +data )
  • pre 在需要使用很多空格是有 pre 标签包裹
  • code 需要写代码时使用
  • hr 分割线
  • br 换行
  • a 超文本链接 注意：target=”_blank” 时，重新开窗口打开。
  • em 强调 语气强调
  • strong 重要，强调事情
  • quote 引用内联
  • blockquote 块级引用
