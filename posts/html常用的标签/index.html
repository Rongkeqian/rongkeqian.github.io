HTML 重点标签

1、<a>标签

  • 作用：

    • 跳转外部页面
    • 跳转内部锚点
    • 跳转到邮箱或者电话

  • 属性

    • href :超文本链接
    • traget：在哪个窗口打开链接
    • download ：忘记好了，现在也没有用
    • rel=noopener:：打开一个外部链接（涉及安全问题）

  • a 的 href 的取值

    • 网址：HTTPS://google.com 跳转到 google
    • 网址：HTTP：//google.com 跳转到 google
    • w 网址：//google.com 浏览器自动选择适用的网址（有没有更高级）

  • 路径

  • 伪协议

  • a 的 target 的取值：

    • _blank:空白的页面打开
    • _top:顶级窗口打开
    • _parent:当前窗口的父级窗口打开
    • _self:自己的窗口打开（默认值）

  • 程序员命名

2、<table>标签

  • 相关的标签
    • table
    • thead
    • tbody
    • tfoot
    • tr //table row 行
    • th 表头
    • td 数据
  • 相关的样式
    • table-layout
    • border-collapse //是否合并边框
    • border-spacing //相邻单元格的间隙

3、<img>标签

  • 作用：发出 get 请求，展示一张图片

  • 属性：

    • alt:对图片的描述，当图片无法显示时，显示的文本。
    • src:图片的路径
    • width 和 height：图片的宽高（前端的底线，不要让一张图片拉伸）
    • max-width：100% 宽度 100%的自适应

4、<tformable>表单标签

  • 作用：发出 get 或 post 请求，然后刷新页面。
  • 属性：
    • action：请求打开哪个页面
    • autocomplete: 自动建议
    • method：控制用 get 还是 post
    • target：规定在何处打开 action 的 URL

5、<input>标签

  • 作用：让用户输入内容

  • 属性：

    • 类型 type：

      1.button (type="button"和直接用<&lt;button&gt;的区别在，input 是一个单标签，无法添加其他标签，button 可以添加其他标签)

2.checkbox （让一组的 CheckBox 拥有同一个 name）

3.radio (注意：单选男女时，设置 name 相同，如：name="gender")

![截图](/input1.png)

4.text

5.submit

6.file (需要上传多个文件添加属性 multiple)

7.hidden

      还有很多其他不常用的标签，需要是可查资料

6、其他标签

  • textarea 多行文本 （此文本是可以拖动的，可通过 css 来设置 resize:none）
  • select 下拉菜单

注意事项：

  • 一般不监听 input 的 click 事件
  • form 里面的 input 要有 name
  • form 里要放一个 type=submit 才能触发 submit 事件
