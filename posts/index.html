如何用hugo搭建博客

如何用hugo搭建博客 搭建博客花了我挺多时间的，因为出错了几次，路径问题，还有最最低级的错误是命令写错（汗颜）！到现在，我都还不是很清楚搭建博客跟我后面的学习有什么关系。先不管了，老师说的都是对滴！！ 下面写写我搭建博客的步骤： 1.首先，先下载hugo并运行命令hugo version(查看是否安装成功)。 2.添加path路径。我的电脑–属性–高级设置–环境变量–path–添加 3.然后进入到hugo首页，点击Quick start。 4.从step2开始抄代码到step7,中间有几个注意的点，如下： step4 hugo new posts/my-first-post.md 中的my-first-post改为自己的名字，进入代码页，需要把draft:true改为draft:false。 执行到step5不要关闭，关了就进不去了，可以另开有一个终端。 step7 当前目录执行hugo -D ，得到一个新的public目录。 5.在当前目录创建.gitignore(忽略文件)，把public写入(/public/)到.gitignore。 如何上传到网上 1.进入到public目录，建立库（git init）、添加库（git add）、提交（git commit -v）。 2.去github新建一个仓库,命名规则是：必须是github的用户名.github.io. 3.然后执行ssh里的两行代码。 4.在settings里GitHub Pages下面的链接就可以进入了。 如何创建第二篇博客 1、在 xxx.github.io-creator 目录（注意确保自己不在 public 目录）里运行 hugo new posts/第二篇博客.md 2.运行 code posts/第二篇博客.md 对文件进行编辑，注意不要把文件原本的内容给删了，直接在后面另起一行写新内容。 3、运行 hugo -D，得到新的 public 目录 4、进入 public 目录，执行一下操作 git add . 注意有一个点 git commit -m update git push -f 其中 -f 是强制上传的意思 等待几分钟后，你的博客就会出现第二篇文章了！
